# fix-bugs-and-usability

Audit pass on the Exquisite Stories cube app (public/). Each child stitch is a discrete bug or UX improvement found during review. Tie children individually as fixes ship.

Severity tags inside each child:
- **bug** — wrong behavior, crash, or broken-on-some-platform
- **ux** — works, but the user experience could be better
- **cleanup** — dead/confusing code, no user impact
